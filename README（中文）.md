从「研究」 → 「结构」 → 「可用系统」

目标只有一个：

让 AI 的高级能力，变得**可用、可控、可复用**。

## 🚀 项目简介

TalkWeb 是一个基于 Agent 的 AI 操作系统，支持：

- 自动选择 Agent  
- 自动生成 Script（提示词）  
- PromUp 提示词优化  
- Skill 技能模块系统  
- 多语言支持  
- Mermaid 图表生成  
- JSON 导入 / 导出  

---

## ✨ 核心能力

### 🤖 Agent 系统
- 自动匹配最合适的 Agent  
- 支持自定义角色  

### 🧠 Skill & Script
- 技能模块化  
- Prompt 模板复用  

### ⚡ PromUp 引擎
- 自动优化输入  
- 提升输出质量  
- 输出结构化结果  

### 🌍 多语言支持
- 中文 / 英文 / 韩文  

### 🔑 无 API 模式
- 支持本地使用  
- 可接入不同 AI 模型  

### 📊 Mermaid 图表
- 自动生成流程图  
- 支持编辑  

### 🎭 人设系统
- 可定义 AI 风格与语气  

### 📦 Prompt 管理
- JSON 一键下载  
- 支持上传 / 本地导入导出
