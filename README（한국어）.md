## ✨ 서문

AI는 이미 매우 강력합니다.  
대부분의 사람들이 생각하는 것보다 훨씬 더 강력합니다.

하지만 중요한 것은 모델 자체가 아니라,  
당신이 그것을 어떻게 사용하는가입니다.

당신은 AI를 단순한 도구로 보나요?  
아니면 당신을 이해할 수 있는 존재로 보나요?

사람 사이에서도 마찬가지입니다.  
같은 말이라도 어떤 사람은 이해하고, 어떤 사람은 이해하지 못합니다.

AI도 동일합니다.

다음이 주어져야 합니다:
- 명확한 의도  
- 충분한 맥락  
- 구조화된 규칙  

그래야 AI는 원하는 결과를 만들어냅니다.

그래서 TalkWeb은 단순한 소프트웨어가 아닙니다.

이는 **AI가 당신을 더 잘 이해하도록 돕는 인터페이스**입니다.

ChatGPT, Gemini, Qwen, Claude 등  
어떤 모델을 사용하든 상관없습니다.

TalkWeb은 당신에게 가장 자연스럽고 편안한入口가 됩니다.

당신은 AI를:
- 하나의 Agent로 만들 수 있고  
- 특정 성격과 톤을 부여할 수 있으며  
- 역할 기반 전문가로 설정하거나 (예: IT, 운영 등)  
- 개성 있는 캐릭터로 만들 수도 있습니다  

지금 바로 시도해보세요.  
당신이 원하는 것을 말하세요.  
함께 발전해봅시다.

## 🔬 연구 기반

TalkWeb은 완전히 처음부터 만들어진 시스템이 아닙니다.

다음과 같은 기반 위에서 만들어졌습니다:

- 대규모 프롬프트 엔지니어링 연구 (구조화된 프롬프트 방법론 포함)  
- GitHub 등 커뮤니티에서 공유된 개발자 및 연구자의 지식  

하지만 이러한 정보들은 보통:
- 흩어져 있고  
- 바로 사용하기 어렵습니다  

그래서 이를 다시 정리하고 구조화하여  
**누구나 바로 사용할 수 있는 시스템**으로 만들었습니다.

연구 → 구조 → 실사용 시스템

목표는 단 하나입니다:

AI의 고급 활용을  
**누구나 사용할 수 있도록 만드는 것**입니다.

## 🚀 개요

TalkWeb은 Agent 기반 AI 운영 시스템으로, 다음을 지원합니다:

- 자동 Agent 선택  
- 자동 프롬프트 생성  
- PromUp (프롬프트 최적화 엔진)  
- Skill 기반 모듈 실행  
- 다국어 지원  
- Mermaid 다이어그램 생성  
- JSON 가져오기 / 내보내기  

---

## ✨ 주요 기능

### 🤖 Agent 시스템
- 사용자 정의 Agent 생성  
- 요청 자동 라우팅  

### 🧠 Skill & Script
- 모듈형 Skill 구조  
- 재사용 가능한 프롬프트 템플릿  

### ⚡ PromUp 엔진
- 입력 자동 최적화  
- 더 명확하고 구조적인 출력 생성  

### 🌍 다국어 지원
- 중국어 / 영어 / 한국어  

### 🔑 API 없이 사용 가능
- 로컬 실행 지원  
- 다양한 AI 모델과 호환  

### 📊 Mermaid 다이어그램
- 워크플로우 자동 생성  
- 결과 수정 가능  

### 🎭 AI 성격 설정
- 톤 및 스타일 정의 가능  

### 📦 프롬프트 관리
- JSON 다운로드  
- 업로드 / 가져오기 / 내보내기
